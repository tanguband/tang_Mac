Mac版 短愚蛮怒 18.0.0 （動作環境は Mac OS X 10.11 以降）

githubにてoddstrさんが公開されている変愚蛮怒 2.0.0 Mac OS X フロントエンドを改変して
短愚蛮怒で使用できるようにした後バイナリビルドしたものです。
この場を借りてoddstrさんに感謝の意を表明いたします。

コンパイル済アプリケーションについて
　~/Documents/Tanguband 以下に保存ディレクトリを勝手に作ります。（変更不可）
　~/Library/Preferences/jp.sourceforge.hengband.Tanguband.plist
　にGUI関連の初期設定が保存されます。
　短愚蛮怒の lib/ は Tanguband.app/Contents/Resources 以下にあります。
　Finder で右クリック→パッケージの内容を表示して下さい。

未サポートの機能
・IMEによる日本語入力（ペーストは可能）
・テキストのコピー
・グラフィックタイル
・サウンド
・保存ディレクトリの変更

既知の問題
・小さい文字でアンチエイリアスが見づらい
・64bit モードでの動作が不安定
・フォントの種類により バックスラッシュ "\" の出力不可
　（替わりに "¥" が出力されますがゲーム内では "\" で認識、マクロで使用可）

ビルド・動作の確認は OSX 10.11.6 （Xcode 8.2） で行なっています。
